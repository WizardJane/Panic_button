# Panic_button
Практикум цифрового производства

Файлы в проекте:

esp32testMIPTFabCommented - программа для подключения mqtt и установки сервера на wi-fi с дополнительного занятия на фабрике по теме "Введение в интернет вещей" (https://mipt-factory.gitbook.io/vvedenie-v-internet-veshei)

dumb_mic - программа для проверки работоспособности микрофона при помощи приложения DumbDisplay на телефоне

just_button - программа для проверки работоспособности кнопки

formatting - программа для форматирования внутренней памяти устройства (SPIFFS)

reading - программа для чтения файла recording.wav из SPIFFS с выводом символов на экран

recording: recording.ino - программа из туториала (https://rutube.ru/video/489ef2116c656f810ce4f21f0124e8f2/?r=wd) по считыванию данных с микрофона. Основа рабочего кода.

my_gps - программа для работы с gps-модулем. Взято из интернета 

reading_28 - программа для записи звука с правильным выделением памяти

reading_29 - усовершенствование reading_28: добавлена кнопка

reading_30 - усовершенствование reading_29: добавлено wi-fi соединение

reading_31 - финальная версия кода

схема.png - схема расположения компонентов устройства на плате

node-red_flows.txt - программа для сервера (формат json)
